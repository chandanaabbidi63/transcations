The dataset contains transaction-level information where each row represents a single purchase. It includes fields such as:

Product Details: product name, type, and category

Service Category: service associated with the purchase

Transaction Data: amount, date, and payment method

This structure allows analysis across multiple business dimensions including geography, time, product performance, and customer behavior.
